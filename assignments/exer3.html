# Week 4 In-class Exercise: Information architecture

# Details

# Create an Information Architecture (20 mins)

Using what you learned in this week's lesson, create an information architecture for Winterlude.

  • It must contain the following content:
    • List of Events (schedule view, alphabetical list, sorted by category)
    • Event details
    • List of locations
    • Map view of locations
    • What's on now
    • Official hotels
    • Official restaurants
    • Shuttle information (stops, routes, schedules)
    • General information
    • Sponsors
    • Canal Services
    • Canal ice conditions

# Deliverable for this exercise:

  • Information Architecture Diagram as a Jpg/png/pdf file
  • Filename: FirstName_lastName_IAdiagram.jpg (or PNG or PDF)

Note

  • Do not just copy the information architecture for the existing Winterlude (opens new window) website. It's not very good, and either way, that's plagiarism.
  • Use any tool you wish for the IA (Visio, FigJam, Illustrator, miro (opens new window), mural (opens new window), moqups (opens new window), pen & paper)
