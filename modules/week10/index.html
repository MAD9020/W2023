# Week 10: Layout, input methods, controls

Learning Objective

By the end of this week, students will:

  • Understand input on mobile devices
  • Know common mobile controls, components, input methods, and when to use them
  • Have an introduction to layout principles

In-class

  • Lecture
  • Introduce forms assignment
  • In-class exercise - Compare input methods
